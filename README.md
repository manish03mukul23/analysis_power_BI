Airtel Africa Telecom Growth Analysis (Power BI Project)
Project Overview

This project analyzes Airtel Africa telecom performance using Power BI. The dashboard shows subscriber growth, revenue trends, ARPU, data users, and network expansion.

The goal of this project is to convert raw data into meaningful business insights using data analytics tools.

Tools Used

• Power BI
• Power Query
• DAX
• Excel

What I Did

• Collected data from multiple sources
• Cleaned and transformed data
• Created data model
• Created DAX measures
• Built interactive dashboard
• Generated insights

Key Insights

• Subscriber growth increased over time
• Revenue increased with network expansion
• 4G and 5G increased data usage
• ARPU shows stable performance

KPIs

• Total Revenue → 6.21 Billion
• Total Subscribers → 1.65 Billion
• Average ARPU → 4.00
• Countries Covered → 13
